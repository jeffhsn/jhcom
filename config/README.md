# Configuration Files

This folder contains backup copies of code quality configuration files.

**Note:** The actual config files must remain at the project root for tool compatibility:
- `.prettierrc` - Prettier formatting configuration
- `.eslintrc.cjs` - ESLint linting configuration  
- `.editorconfig` - Editor configuration
- `.prettierignore` - Prettier ignore patterns

These files are kept here for reference and version control purposes.
